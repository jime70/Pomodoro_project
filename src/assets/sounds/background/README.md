# Carpeta de Música de Fondo

Coloca aquí tus archivos de audio para la música de fondo durante las sesiones de estudio.

## Archivos actuales:

- `piano-romantic.mp3` - Piano romántico
- `piano-rain.mp3` - Piano con lluvia
- `rain-ambience.mp3` - Ambiente de lluvia
- `forest-birds.mp3` - Sonidos de bosque con pájaros
- `summer-memories.wav` - Recuerdos de verano

## Notas:

- Los archivos pueden ser `.mp3` o `.wav`
- Todos los archivos se reproducirán en loop durante las sesiones de estudio
- El código funcionará con cualquier formato de audio soportado por el navegador
